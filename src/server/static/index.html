치지직 개발자 등록

이렇게 하세요:

  1. 치지직 개발자센터에 접속해서 로그인하세요
  2. "새 애플리케이션 등록"을 눌러주세요
  3. 앱 이름은 아무거나 (예: "내 후원 연동")
  4. 로그인 리다이렉트 URL에 http://localhost:3000/auth/callback 을 입력하세요
  5. 등록하면 나오는 Client ID와 Client Secret을 아래에 붙여넣기 하세요
치지직 내 채널 페이지 URL의 마지막 부분이에요

VTuber Studio 연결

VTuber Studio에서 이렇게 하세요:

  1. VTuber Studio를 실행하세요

  3. "API (Plugin) 사용"을 켜기로 바꿔주세요
  4. 잠시 기다리면 "Chzzk Donation Bridge" 플러그인 승인 팝업이 떠요
  5. 허용을 눌러주세요
